# Dataset

This project uses a custom dataset of **3,480 annotated images**
captured under adverse weather conditions.

Dataset distribution:

• YOLOv10
  - Train: 915
  - Validation: 261
  - Test: 131

• VGG19
  - Train: 1175
  - Validation: 336
  - Test: 168

Images were collected from YouTube videos and annotated using Roboflow.

The dataset is not publicly available.
